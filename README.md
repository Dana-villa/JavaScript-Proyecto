🎓 Dashboard de Estudiante - Proyecto Académico
Este es un panel de control interactivo diseñado para estudiantes de la Institución Educativa ABC. El proyecto se enfoca en proporcionar una experiencia de usuario moderna (UX) y una interfaz limpia (UI) para la gestión de cursos, progreso académico y notificaciones.

🚀 Características Principales
Dashboard Dinámico: Resumen de actividad, cursos inscritos y progreso académico en tiempo real.

Sistema de Modales (Ventanas Flotantes): Arquitectura de ventanas emergentes personalizadas para ver perfiles de usuario y detalles de materias sin recargar la página.

Barra Lateral Interactiva: Navegación fluida entre el panel principal y la gestión de cursos.

Diseño Adaptable (Responsive): Optimizado para diferentes tamaños de pantalla.

Gestión de Notificaciones: Sistema visual para recordatorios de exámenes y tareas pendientes.

🛠️ Tecnologías Utilizadas
HTML5: Estructura semántica de la plataforma.

CSS3: Diseño personalizado con Flexbox, CSS Grid y animaciones de transición.

JavaScript (Vanilla): Lógica de manipulación del DOM, gestión de eventos y motor de modales dinámicos.

Google Fonts: Tipografía Inter para una lectura clara.
